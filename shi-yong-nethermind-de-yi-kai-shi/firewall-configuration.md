# 防火墙配置

{% hint style="info" %}
除非受信任的机器外，保持`8545` \(or `8000-9000`\) 关闭，这是JSON RPC端口，它允许管理员访问节点。
{% endhint %}

{% hint style="info" %}
保持TCP 30303 \(`30300-30400`\)对外开放，以便其他节点可以连接到您的节点。
{% endhint %}

{% hint style="info" %}
保持UDP `30303` \( 或`30300-30400`\) 对外开放，以便其他节点可以发现您的节点。
{% endhint %}
