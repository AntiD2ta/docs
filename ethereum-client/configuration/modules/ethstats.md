# EthStats

| 属性名称 | 描述 | 默认值 |
| :--- | :--- | ---: |
| `Contact` | ethstats 页面上显示的节点所有者的联系方式。 | `null` |
| `Enabled` | 如果被设置为 `true`，则启用 EthStats 发布。 | `false` |
| `Name` | 指定 ethstats 服务器上显示的节点名称。 | `null` |
| `Secret` | 发布到指定 ethstats 服务器所使用的密码。 | `null` |
| `Server` | EthStats 服务器 `wss://hostname:port/api/` 。 | `null` |
