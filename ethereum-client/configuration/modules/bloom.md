# Bloom

| 属性名称 | 说明 | 默认值 |
| :--- | :--- | ---: |
| `Index` | 定义是否使用Bloom索引。 Bloom索引加快，rpc日志搜索的速度。 | `null` |
| `IndexLevelBucketSizes` | 定义索引级别的乘数。可以对每个链进行调整以提高性能。 | `[4, 8, 8]` |
| `Migration` | 定义索引级别的乘数。可以对每个链进行调整以提高性能。 | `false` |
| `MigrationStatistics` | 定义索引级别的乘数。可以对每个链进行调整以提高性能。 | `false` |
