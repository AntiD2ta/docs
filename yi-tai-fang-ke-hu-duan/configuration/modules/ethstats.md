# EthStats

| 属性名称 | 说明 | 默认值 |
| :--- | :--- | ---: |
| `Contact` | ethstats页面上显示的有节点人的联系详细信息。 | `null` |
| `Enabled` | 如果为`true` 启用EthStats发布 | `false` |
| `Name` | 显示在给定ethstats服务器上的节点名称。 | `null` |
| `Secret` | 发布到给定的ethstats服务器的密码。 | `null` |
| `Server` | EthStats服务器`wss://hostname:port/api/` | `null` |
