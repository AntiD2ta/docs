# Diag module

## diag.cliVersion

Displays client version

| Invocación |
| :--- |
| `diag.cliVersion` |

| este método no tiene parametros  |
| :--- |


| Tipo de retorno | Descripción |
| :--- | :--- |
| `JavaScript Object` | Client version |
